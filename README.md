Calculadora interativa de gestão para transportadoras, desenvolvida como projeto prático de **Matemática Computacional** na FATEC Zona Sul. Aplica conceitos matemáticos reais com linguagem simples e resultados explicados de forma prática.

---

## 🧮 Ferramentas

| Ferramenta | O que faz |
|---|---|
| 💹 Reinvestimento de Lucros | Simula crescimento do lucro com juros compostos |
| 🚛 Financiamento de Caminhão | Calcula total de juros simples no financiamento |
| ⚖️ Ponto de Equilíbrio | Quantos fretes por mês para cobrir os custos |
| 🗺️ Otimização de Rotas | Em quais distâncias o custo da rota se equilibra |
| ✅ Viabilidade de Frete | Distância mínima para o frete ser lucrativo |
| 📊 Faixa de Lucratividade | Intervalo de KM onde o frete dá lucro |
| 🔗 Operações com Rotas | Organiza clientes em rotas com teoria dos conjuntos |
